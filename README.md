## 📖 Overview

CMSIS-compliant header files for the Texas Instruments TM4C1294NCPDT microcontroller. Includes register mappings, interrupt vector definitions, and peripheral structures for GPIO, UART, SPI, I2C, ADC, Timer, PWM, and more.

## 📚 Peripherals Covered

| Peripheral | Description |
|:-----------|:------------|
| GPIO | Port A-Q, alternate function, interrupt |
| UART | 8 UART modules with FIFO |
| SPI/SSI | 4 synchronous serial interfaces |
| I2C | 4 I2C modules |
| ADC | 2 ADC modules, 20 channels |
| Timer | 8 GPTM, wide timers, PWM |
| CAN | 2 CAN controllers |
